**Bilingual Reference Notes (English / Português)**

---

## Question 1  
**EN:** A cloud security analyst has a collection of projects and other materials they can share with interviewers to demonstrate their skills. What is the term for this collection?  
**PT:** Um analista de segurança em nuvem possui uma coleção de projetos e outros materiais que pode compartilhar com entrevistadores para demonstrar suas habilidades. Qual é o termo para essa coleção?

- Portfolio / Portfólio ✅  
- Network / Rede  
- Biography / Biografia  
- Framework / Estrutura  

**Explanation / Explicação:**  
EN: A portfolio is a collection of materials that demonstrates technical and analytical skills to potential employers.  
PT: Um portfólio é uma coleção de materiais que demonstra habilidades técnicas e analíticas para potenciais empregadores.  

---

## Question 2  
**EN:** After completing a final interview, what step should a cloud security analyst take next?  
**PT:** Após concluir a entrevista final, qual passo o analista de segurança em nuvem deve tomar em seguida?

- Send a thank you message / Enviar uma mensagem de agradecimento ✅  
- Research company mission / Pesquisar a missão da empresa  
- Submit portfolio / Enviar portfólio  
- Recruiter call / Chamada com recrutador  

**Explanation / Explicação:**  
EN: Following up with a thank-you message shows professionalism and keeps communication open.  
PT: Enviar uma mensagem de agradecimento demonstra profissionalismo e mantém a comunicação aberta.  

---

## Question 3  
**EN:** How can an analyst include confidential project experience in a portfolio without violating policies?  
**PT:** Como um analista pode incluir experiência em projetos confidenciais em um portfólio sem violar políticas?

- Basic description of project and role / Descrição básica do projeto e papel ✅  
- Encrypted repository / Repositório criptografado  
- Code sample / Amostra de código  
- Employer-hosted site / Site hospedado pelo empregador  

**Explanation / Explicação:**  
EN: A general description of the project and role highlights skills without exposing confidential data.  
PT: Uma descrição geral do projeto e do papel destaca habilidades sem expor dados confidenciais.  

---

## Question 4  
**EN:** Which two characteristics should a portfolio include?  
**PT:** Quais duas características um portfólio deve incluir?

- Biography / Biografia ✅  
- Visually appealing design / Design visual atrativo ✅  
- Copyrighted materials / Materiais protegidos  
- Job history / Histórico de empregos  
- List of technical skills / Lista de habilidades técnicas  

**Explanation / Explicação:**  
EN: A biography provides context, while a visually appealing design makes the portfolio easier to navigate.  
PT: A biografia fornece contexto, enquanto um design visual atrativo torna o portfólio mais fácil de navegar.
