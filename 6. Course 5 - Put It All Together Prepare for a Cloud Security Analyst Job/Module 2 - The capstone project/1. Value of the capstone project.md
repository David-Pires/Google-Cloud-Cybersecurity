

# 🎓 The Capstone Project: Showcase Your Expertise

# 🎓 O Projeto Capstone: Demonstre sua Especialidade

O Projeto Capstone é o fechamento estratégico do curso, onde você aplicará de forma integrada os conhecimentos de IAM, Redes, Detecção de Ameaças e BCDR.

---

## 🏗️ O que esperar do Capstone / What to expect

O projeto simula um cenário real de segurança em nuvem, exigindo que você:

1. **Analise um Cenário de Negócio:** Identifique riscos e requisitos de conformidade.
2. **Implemente Controles de Segurança:** Configure VPCs, Firewalls e políticas de IAM.
3. **Monitore e Responda:** Utilize o SCC e Cloud Logging para detectar e mitigar incidentes simulados.
4. **Garanta a Resiliência:** Aplique métricas de RPO/RTO e planos de recuperação (DR).

---

## 💡 Dicas para o Sucesso / Tips for Success

* **🇺🇸 Documentation is Key:** Don't just do the technical work; document the "why" behind every decision. This shows hiring managers your consultative and architectural thinking.
* **🇧🇷 Documentação é Fundamental:** Não faça apenas o trabalho técnico; documente o "porquê" de cada decisão. Isso mostra aos recrutadores seu pensamento consultivo e arquitetural.
* **🇺🇸 Connect to Prior Experience:** Use your background in Infrastructure to explain how cloud security solutions improve upon traditional on-premises methods.
* **🇧🇷 Conecte com Experiência Prévia:** Use sua bagagem em Infraestrutura para explicar como as soluções de segurança em nuvem superam os métodos tradicionais locais.

---

## 💼 O Capstone no Processo Seletivo / The Capstone in the Job Search

Este projeto será o seu principal tópico de conversa em entrevistas técnicas:

* **No Currículo:** Adicione uma seção "Destaque de Projeto" descrevendo o Capstone.
* **No Portfólio:** Publique o código (Terraform/CLI) e o relatório final no GitHub.
* **Na Entrevista:** Use o Capstone como base para o método **STARL**. Ele será a sua "Situação" e "Tarefa" prontas para serem explicadas.
