# 🔍 Técnicas de Pesquisa de Alertas | Alert Search Techniques

**🇺🇸 English:** Querying alerts is the process of scanning through security notifications to pinpoint real threats. Effective search techniques reduce "noise" and allow the security team to focus on high-impact incidents.
**🇧🇷 Português:** Pesquisar alertas é o processo de analisar notificações de segurança para identificar ameaças reais. Técnicas de pesquisa eficazes reduzem o "ruído" e permitem que a equipa de segurança se foque em incidentes de alto impacto.

---

### 1. Técnicas para Pesquisas Eficazes | Effective Querying Techniques

Para evitar a sobrecarga, os analistas utilizam filtros e parâmetros específicos para refinar os resultados:

| 🇺🇸 Technique | 🇧🇷 Técnica | 💡 Objetivo / Objective |
| --- | --- | --- |
| **Severity Filtering** | **Filtragem por Severidade** | 🇺🇸 Focus on "High" and "Critical" alerts first. <br>

<br> 🇧🇷 Focar primeiro em alertas "Altos" e "Críticos". |
| **Timeframe Analysis** | **Análise Temporal** | 🇺🇸 Correlate alerts that happened within the same window. <br>

<br> 🇧🇷 Correlacionar alertas que ocorreram na mesma janela de tempo. |
| **Category Grouping** | **Agrupamento por Categoria** | 🇺🇸 Isolate specific types, like IAM changes or VPC flow logs. <br>

<br> 🇧🇷 Isolar tipos específicos, como mudanças de IAM ou logs de VPC. |

---

### 2. Melhores Práticas | Best Practices

**🇺🇸 English:** Developing a methodology for searching alerts is key to maintaining a resilient security posture.
**🇧🇷 Português:** Desenvolver uma metodologia para pesquisar alertas é fundamental para manter uma postura de segurança resiliente.

* **🇺🇸 Specificity:** Make searches as narrow as possible (e.g., search by Source IP or specific Service Account) to reduce distractions.
* **🇧🇷 Especificidade:** Torne as pesquisas o mais restritas possível (ex: pesquisar por IP de origem ou conta de serviço específica) para reduzir distrações.
* **🇺🇸 Saved Queries:** Save frequently used search patterns to speed up future investigations.
* **🇧🇷 Consultas Guardadas:** Guarde padrões de pesquisa frequentes para acelerar investigações futuras.
* **🇺🇸 Regular Reassessment:** Update filters regularly to adapt to new cloud services or evolving threat patterns.
* **🇧🇷 Reavaliação Regular:** Atualize os filtros regularmente para se adaptar a novos serviços de nuvem ou padrões de ameaças em evolução.

---

### 3. O Fluxo de Triagem | The Triage Workflow

**🇺🇸 English:** When querying, the analyst follows a logical flow: **Filter → Prioritize → Investigate → Resolve.**
**🇧🇷 Português:** Ao pesquisar, o analista segue um fluxo lógico: **Filtrar → Priorizar → Investigar → Resolver.**

1. **Filtrar (Filter):** Remover alertas conhecidos como benignos ou de baixa prioridade.
2. **Priorizar (Prioritize):** Atacar primeiro os alertas que envolvem dados sensíveis ou permissões administrativas.
3. **Investigar (Investigate):** Cruzar o alerta com os logs brutos para entender o contexto total.
4. **Resolver (Resolve):** Aplicar a remediação e atualizar a regra de alerta para evitar falsos positivos futuros.

## 🛡️ Insight do Analista | Analyst's Insight

**🇺🇸 English:**
Mastering **Boolean logic** (AND, OR, NOT) and specialized query languages (like **YARA-L** in Google Chronicle) is a superpower. Instead of looking at "all login alerts," a pro queries "Logins AND NOT from Corporate VPN AND Severity: High." This reduces thousands of results to a handful of critical leads.

**🇧🇷 Português:**
*Dominar a **lógica Booleana** (AND, OR, NOT) e linguagens de consulta especializadas (como **YARA-L** no Google Chronicle) é um superpoder. Em vez de olhar para "todos os alertas de login", um profissional pesquisa "Logins AND NOT de VPN Corporativa AND Severidade: Alta". Isso reduz milhares de resultados para apenas alguns indícios críticos.*

### 🚀 Glossário Final de SecOps | Final SecOps Glossary

* **🇺🇸 Query:** A request for information from a database or search engine.
* **🇧🇷 Consulta (Query):** Um pedido de informação a uma base de dados ou motor de busca.
* **🇺🇸 False Negative:** A security threat that is NOT flagged by the system (the most dangerous scenario).
* **🇧🇷 Falso Negativo:** Uma ameaça de segurança que NÃO é sinalizada pelo sistema (o cenário mais perigoso).
* **🇺🇸 SIEM Tuning:** The process of adjusting rules to improve alert accuracy.
* **🇧🇷 Ajuste de SIEM (Tuning):** O processo de ajustar regras para melhorar a precisão dos alertas.

**"The faster you can query, the faster you can contain. Speed is the best friend of a security analyst."**
*(Quanto mais rápido puder pesquisar, mais rápido poderá conter. A velocidade é a melhor amiga de um analista de segurança.)*
