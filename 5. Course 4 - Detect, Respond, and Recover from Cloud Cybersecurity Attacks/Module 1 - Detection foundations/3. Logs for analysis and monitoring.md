# 📜 Logs para Análise e Monitoramento | Logs for Analysis & Monitoring

**🇺🇸 English:** Logs are historical records of events within a system. Log management is the process of collecting, storing, and analyzing this data to identify security incidents and maintain compliance.
**🇧🇷 Português:** Logs são registros históricos de eventos dentro de um sistema. A gestão de logs é o processo de coletar, armazenar e analisar esses dados para identificar incidentes de segurança e manter a conformidade.

---

### 1. Tipos de Logs no Google Cloud | Types of Logs in Google Cloud

O Google Cloud categoriza os logs de auditoria para facilitar a triagem e a conformidade:

| 🇺🇸 Log Type | 🇧🇷 Tipo de Log | 💡 What it tracks / O que rastreia |
| --- | --- | --- |
| **Admin Activity** | **Atividade de Administrador** | 🇺🇸 Actions like creating databases or changing IAM permissions. <br>

<br> 🇧🇷 Ações como criar bancos de dados ou alterar permissões de IAM. |
| **Data Access** | **Acesso a Dados** | 🇺🇸 API calls that read or modify user-provided data (Who saw what?). <br>

<br> 🇧🇷 Chamadas de API que leem ou modificam dados do usuário (Quem viu o quê?). |
| **System Event** | **Eventos de Sistema** | 🇺🇸 Administrative actions that change system configurations. <br>

<br> 🇧🇷 Ações administrativas que alteram as configurações do sistema. |
| **Policy-Denied** | **Negação de Política** | 🇺🇸 Attempts to access resources that were blocked by IAM policies. <br>

<br> 🇧🇷 Tentativas de acessar recursos que foram bloqueadas por políticas de IAM. |

### 2. Ciclo de Vida e Retenção | Log Life Cycle & Retention

**🇺🇸 English:** Logs follow a strict lifecycle: **Generation → Storage → Analysis → Expiration**. Organizations must balance storage costs with compliance requirements when defining retention periods.
**🇧🇷 Português:** Os logs seguem um ciclo de vida rigoroso: **Geração → Armazenamento → Análise → Expiração**. As organizações devem equilibrar os custos de armazenamento com os requisitos de conformidade ao definir os períodos de retenção.

* **🇺🇸 Retention:** Crucial for forensic investigations. Some regulations require logs to be kept for years.
* **🇧🇷 Retenção:** Crucial para investigações forenses. Algumas regulamentações exigem que os logs sejam mantidos por anos.

## 🛡️ Insight do Analista | Analyst's Insight

**🇺🇸 English:**
"Policy-Denied" logs are a goldmine for security analysts. While one denied request might be a user mistake, a burst of **Policy-Denied** events from the same IP often indicates an attacker performing **Permissions Discovery** to see what they can access.

**🇧🇷 Português:**
*Os logs de "Negação de Política" (Policy-Denied) são uma mina de ouro para analistas de segurança. Embora uma solicitação negada possa ser um erro do usuário, uma explosão de eventos **Policy-Denied** do mesmo IP geralmente indica um invasor realizando a **Descoberta de Permissões** para ver o que ele consegue acessar.*

### 🚀 Glossário de Termos | Key Terms Glossary

* **🇺🇸 Audit Trail:** A chronological record that provides documentary evidence of the sequence of activities.
* **🇧🇷 Trilha de Auditoria:** Um registro cronológico que fornece evidência documental da sequência de atividades.
* **🇺🇸 SIEM (Security Information and Event Management):** Software that provides real-time analysis of security alerts generated by applications and network hardware.
* **🇧🇷 SIEM:** Software que fornece análise em tempo real de alertas de segurança gerados por aplicações e hardware de rede.

**"In the cloud, logs are the only truth. If it's not logged, it didn't happen."**
*(Na nuvem, os logs são a única verdade. Se não foi registrado, não aconteceu.)*
