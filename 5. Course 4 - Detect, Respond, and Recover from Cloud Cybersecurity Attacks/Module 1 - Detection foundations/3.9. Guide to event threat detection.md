
# 🛡️ Event Threat Detection & SCC | Guia de Detecção de Ameaças

**🇺🇸 English:** Event Threat Detection is a built-in service of the Security Command Center (SCC) that provides log-based threat analysis. It continuously monitors Google Cloud logs to detect potential threats and displays findings centrally.
**🇧🇷 Português:** O Event Threat Detection é um serviço integrado do Security Command Center (SCC) que fornece análise de ameaças baseada em logs. Ele monitoriza continuamente os logs do Google Cloud para detetar potenciais ameaças e exibe as descobertas centralmente.

---

### 1. Security Command Center (SCC)

**🇺🇸 English:** SCC is the centralized vulnerability and threat reporting service for Google Cloud. It scans resources for security issues, vulnerabilities, and misconfigurations.
**🇧🇷 Português:** O SCC é o serviço centralizado de relatórios de vulnerabilidades e ameaças do Google Cloud. Ele faz o varrimento de recursos em busca de problemas de segurança, vulnerabilidades e erros de configuração.

* **🇺🇸 Premium Tier:** Required for full access to advanced built-in services like Event Threat Detection.
* **🇧🇷 Camada Premium:** Necessária para acesso total a serviços integrados avançados, como o Event Threat Detection.

---

### 2. Investigação com Logs Explorer | Investigation with Logs Explorer

**🇺🇸 English:** When a threat is detected, analysts use the **Logs Explorer** to add context to the findings. A common investigation involves monitoring unauthorized IAM changes.
**🇧🇷 Português:** Quando uma ameaça é detetada, os analistas usam o **Logs Explorer** para adicionar contexto às descobertas. Uma investigação comum envolve a monitorização de alterações não autorizadas de IAM.

#### **🇺🇸 Key Query Example | 🇧🇷 Exemplo de Consulta Chave**

```sql
protoPayload.authorizationInfo.permission="resourcemanager.projects.setIamPolicy"
protoPayload.methodName="InsertProjectOwnershipInvite"

```

* **🇺🇸 Logic:** This query searches for attempts to grant project ownership or modify IAM policies, which could indicate an attacker trying to gain full control.
* **🇧🇷 Lógica:** Esta consulta procura tentativas de conceder a posse do projeto ou modificar políticas de IAM, o que pode indicar um invasor tentando obter controlo total.

---

### 3. O Ciclo de Descoberta | The Finding Lifecycle

**🇺🇸 English:** Findings in SCC provide a detailed view of the security event, including the source, the resource affected, and the severity level.
**🇧🇷 Português:** As "Descobertas" (Findings) no SCC fornecem uma visão detalhada do evento de segurança, incluindo a origem, o recurso afetado e o nível de severidade.

1. **🇺🇸 Monitoring:** Event Threat Detection scans logs (Admin Activity, Data Access, etc.).
**🇧🇷 Monitorização:** O Event Threat Detection varre logs (Atividade de Admin, Acesso a Dados, etc.).
2. **🇺🇸 Detection:** A pattern matches a known threat signature.
**🇧🇷 Detecção:** Um padrão corresponde a uma assinatura de ameaça conhecida.
3. **🇺🇸 Reporting:** The finding is published in the SCC Dashboard for analysis.
**🇧🇷 Relatório:** A descoberta é publicada no Painel do SCC para análise.

## 🛡️ Insight do Analista | Analyst's Insight

**🇺🇸 English:**
Event Threat Detection is powerful because it bridges the gap between raw logs and actionable security alerts. Instead of manually searching through millions of log lines, SCC highlights the most critical events, such as a "Project Ownership Invite," which is a high-risk indicator of a potential account takeover.

**🇧🇷 Português:**
*O Event Threat Detection é poderoso porque preenche a lacuna entre logs brutos e alertas de segurança acionáveis. Em vez de pesquisar manualmente por milhões de linhas de log, o SCC destaca os eventos mais críticos, como um "Convite de Propriedade do Projeto", que é um indicador de alto risco de um potencial roubo de conta.*

### 🚀 Glossário Técnico | Technical Glossary

* **🇺🇸 Finding:** A record of a potential security issue detected by SCC.
* **🇧🇷 Descoberta (Finding):** Um registro de um potencial problema de segurança detetado pelo SCC.
* **🇺🇸 Log-based Analysis:** Monitoring system logs to identify malicious patterns.
* **🇧🇷 Análise baseada em Logs:** Monitorizar logs do sistema para identificar padrões maliciosos.
* **🇺🇸 IAM Alert:** A notification triggered by suspicious Identity and Access Management changes.
* **🇧🇷 Alerta de IAM:** Uma notificação disparada por alterações suspeitas na Gestão de Identidades e Acessos.

**"Visibility is the first step of defense. Event Threat Detection ensures that no critical security event goes unnoticed."**
*(Visibilidade é o primeiro passo da defesa. O Event Threat Detection garante que nenhum evento de segurança crítico passe despercebido.)*
