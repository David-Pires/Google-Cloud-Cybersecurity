
# 🔔 Identificação de Alertas Acionáveis | Identifying Actionable Alerts

**🇺🇸 English:** An event is an observed change in normal operations, but not all events require action. An **actionable alert** is a notification that indicates a potential incident—a violation of security policies—that negatively impacts operations and requires a formal response.
**🇧🇷 Português:** Um evento é uma mudança observada nas operações normais, mas nem todos os eventos exigem ação. Um **alerta acionável** é uma notificação que indica um incidente potencial — uma violação das políticas de segurança — que impacta negativamente as operações e exige uma resposta formal.

---

### 1. Do Alerta ao Incidente | From Alert to Incident

Nem toda falha de login é um ataque. O analista deve diferenciar o ruído da ameaça real:

* **🇺🇸 Normal Event:** A few failed logins after a user returns from vacation.
* **🇧🇷 Evento Normal:** Algumas falhas de login após um usuário retornar de férias.
* **🇺🇸 Actionable Alert:** A large number of failed login attempts in a short time from an unfamiliar IP address.
* **🇧🇷 Alerta Acionável:** Um grande número de tentativas de login falhas em pouco tempo a partir de um endereço IP desconhecido.
* **🇺🇸 True Incident:** High network traffic originating from protected data storage by an unauthorized user.
* **🇧🇷 Incidente Real:** Tráfego de rede alto originando-se de armazenamento de dados protegidos por um usuário não autorizado.

---

### 2. O Papel das Métricas | The Role of Metrics

As métricas permitem medir o **"o quê"** e o **"como"** de um evento para identificar problemas rapidamente:

* **O Quê (What):** Ex: Uso elevado de CPU.
* **O Como (How):** Ex: Atividade alta fora do horário comercial regular.

## 🛡️ Insight do Analista | Analyst's Insight

**🇺🇸 English:**
Security tools provide the data, but the **incident response process** is initiated by people. You must document and investigate events early, even if you are not sure they are actionable. By considering each alert's origin, purpose, and impact, you gain the information needed to decide on the appropriate actions.

**🇧🇷 Português:**
*As ferramentas de segurança fornecem os dados, mas o **processo de resposta a incidentes** é iniciado por pessoas. Você deve documentar e investigar eventos cedo, mesmo que não tenha certeza de que são acionáveis. Ao considerar a origem, o propósito e o impacto de cada alerta, você obtém a informação necessária para decidir sobre as ações apropriadas.*

### 🚀 Glossário de Alertas | Alerts Glossary

* **🇺🇸 Actionable Alert | 🇧🇷 Alerta Acionável:** Uma notificação que exige que o profissional de segurança inicie o processo de resposta.
* **🇺🇸 Trigger | 🇧🇷 Gatilho:** Uma condição específica que, quando atendida, gera um alerta (ex: acesso a dados protegidos).
* **🇺🇸 Normal Operations | 🇧🇷 Operações Normais:** O estado padrão de funcionamento dos sistemas sem violações de segurança.
