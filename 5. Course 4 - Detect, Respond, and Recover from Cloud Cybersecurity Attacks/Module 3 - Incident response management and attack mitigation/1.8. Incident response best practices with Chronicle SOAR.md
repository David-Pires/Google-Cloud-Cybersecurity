

# ⚡ Melhores Práticas de Resposta com Chronicle SOAR

**🇺🇸 English:** Chronicle SOAR leverages automation and orchestration to streamline incident response. It uses connectors to ingest data, groups alerts into cases, and executes playbooks to mitigate threats without constant human intervention.
**🇧🇷 Português:** O Chronicle SOAR utiliza automação e orquestração para otimizar a resposta a incidentes. Ele usa conectores para ingerir dados, agrupa alertas em casos e executa playbooks para mitigar ameaças sem a necessidade de intervenção humana constante.

---

### 1. Componentes do Chronicle SOAR | SOAR Components

Para que a automação funcione, o SOAR organiza a informação em uma hierarquia lógica:

* **Connectors (Conectores):**
* **🇺🇸 Role:** The ingestion point that translates raw data from 3rd party tools into normalized SOAR data.
* **🇧🇷 Função:** O ponto de ingestão que traduz dados brutos de ferramentas de terceiros em dados normalizados para o SOAR.


* **Cases, Alerts, and Events:**
* **🇺🇸 Hierarchy:** A **Case** contains **Alerts**, which are made of **Events** (observable occurrences).
* **🇧🇷 Hierarquia:** Um **Caso** contém **Alertas**, que são compostos por **Eventos** (ocorrências observáveis).


* **Entities (Entidades):**
* **🇺🇸 Concept:** Points of interest extracted from alerts (IPs, URLs, file hashes). They are mapped and modeled to provide context to the investigation.
* **🇧🇷 Conceito:** Pontos de interesse extraídos de alertas (IPs, URLs, hashes de arquivos). São mapeados e modelados para fornecer contexto à investigação.



---

### 2. Playbooks: A Árvore de Ações | Playbooks: The Action Tree

**🇺🇸 English:** A playbook is an automated process consisting of a tree of actions. It can be triggered manually or automatically based on predefined criteria, reaching a resolution when the final action is completed.
**🇧🇷 Português:** Um playbook é um processo automatizado que consiste em uma árvore de ações. Ele pode ser disparado manual ou automaticamente com base em critérios predefinidos, alcançando uma resolução quando a última ação é concluída.

**Quando automatizar? (Perguntas de Decisão):**

1. A tarefa é **rotineira**?
2. É uma tarefa **detalhada** e que exige precisão?
3. É uma tarefa que **consome muito tempo** da equipe?

---

### 3. Exemplo Prático: E-mail Suspeito | Practical Use Case

O SOAR não apenas detecta, ele **orquestra** a resposta:

1. **Investigação:** Verifica a reputação do remetente via inteligência de ameaças e DNS.
2. **Extração e Validação:** Extrai hiperlinks e valida a reputação das URLs.
3. **Análise:** Executa o código em um ambiente seguro (sandbox).
4. **Resposta (Playbook):** Se confirmado o incidente, o SOAR busca todas as mensagens do mesmo remetente no sistema e as coloca em **quarentena** automaticamente.

## 🛡️ Insight do Analista | Analyst's Insight

**🇺🇸 English:**
Automation doesn't mean "no humans." One of the best practices for **Chronicle SOAR** is **Data Hygiene**: you must purge what you don't need to keep the system fast. Orchestration is the "brain" that coordinates different tools (Firewall, E-mail, IAM) to act as one. The analyst's new role is to be the **Playbook Designer**, adjusting the automation logic as the threat landscape changes.

**🇧🇷 Português:**
*Automação não significa "sem humanos". Uma das melhores práticas para o **Chronicle SOAR** é a **Higiene de Dados**: você deve descartar o que não precisa para manter o sistema rápido. A orquestração é o "cérebro" que coordena diferentes ferramentas (Firewall, E-mail, IAM) para agirem como uma só. O novo papel do analista é ser o **Designer de Playbooks**, ajustando a lógica de automação conforme o cenário de ameaças muda.*

### 🚀 Glossário de Orquestração | Orchestration Glossary

* **🇺🇸 Ingestion | 🇧🇷 Ingestão:** O processo de importar arquivos de dados de diferentes fontes para um único meio de armazenamento.
* **🇺🇸 SDK (Software Development Kit):** Conjunto de ferramentas (como o SDK em Python do Chronicle) que permite criar novos conectores.
* **🇺🇸 Entities | 🇧🇷 Entidades:** Objetos que representam indicadores (IOCs) ou artefatos extraídos de um alerta.
* **🇺🇸 Orchestration | 🇧🇷 Orquestração:** A habilidade de coordenar decisões e ações responsivas entre múltiplos sistemas de TI.

**"SOAR is about standards; if your process is a mess, automation will only make it a faster mess."**
*(SOAR é sobre padrões; se seu processo é uma bagunça, a automação apenas o tornará uma bagunça mais rápida.)*
