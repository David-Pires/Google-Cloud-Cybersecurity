

# 📊 Agregação e Correlação de Dados | Data Aggregation & Correlation

**🇺🇸 English:** To stay ahead of threats, organizations must collect vast amounts of data from their infrastructure. Aggregation and correlation allow security teams to monitor activities, identify patterns, and detect potential breaches effectively.
**🇧🇷 Português:** Para se antecipar às ameaças, as organizações devem coletar vastas quantidades de dados de sua infraestrutura. A agregação e a correlação permitem que as equipes de segurança monitorem atividades, identifiquem padrões e detectem possíveis invasões de forma eficaz.

---

### 1. Agregação: O "Harvest" de Dados | Aggregation: Data Harvesting

**🇺🇸 English:** Aggregation is the process of gathering and consolidating diverse forms of data from multiple sources (firewalls, IDS, user activities) into a central location.
**🇧🇷 Português:** Agregação é o processo de reunir e consolidar diversas formas de dados de múltiplas fontes (firewalls, IDS, atividades de usuários) em um local central.

* **🇺🇸 Unified View:** Provides a comprehensive view of the entire cloud environment.
* **🇧🇷 Visão Unificada:** Fornece uma visão abrangente de todo o ambiente de nuvem.
* **🇺🇸 Chronicle SIEM:** An example of a tool that aggregates logs from firewalls and Data Loss Prevention (DLP) tools.
* **🇧🇷 Chronicle SIEM:** Um exemplo de ferramenta que agrega logs de firewalls e ferramentas de Prevenção de Perda de Dados (DLP).

---

### 2. Normalização e Correlação | Normalization & Correlation

**🇺🇸 English:** After aggregation, data must be processed to be useful.
**🇧🇷 Português:** Após a agregação, os dados precisam ser processados para serem úteis.

* **🇺🇸 Normalization:** The processing of raw data into a consistent format, which improves correlation performance.
* **🇧🇷 Normalização:** O processamento de dados brutos em um formato consistente, o que melhora o desempenho da correlação.
* **🇺🇸 Correlation:** The relationship between two or more security events. It involves adding context and making connections between events to distinguish threats from legitimate actions.
* **🇧🇷 Correlação:** O relacionamento entre dois ou mais eventos de segurança. Envolve adicionar contexto e fazer conexões entre eventos para distinguir ameaças de ações legítimas.

---

### 3. Exemplo Prático: Login Suspeito | Practical Example: Suspicious Login

**🇺🇸 English:** Correlation is used to cross-reference data points to validate alerts.
**🇧🇷 Português:** A correlação é usada para cruzar pontos de dados e validar alertas.

* **🇺🇸 The Event:** A user logs in from a new device, triggering an alert.
* **🇧🇷 O Evento:** Um usuário faz login de um novo dispositivo, disparando um alerta.
* **🇺🇸 The Correlation:** The system checks the **IP address**, **time of day**, and **past behavior**.
* **🇧🇷 A Correlação:** O sistema verifica o **endereço IP**, o **horário** e o **comportamento passado**.
* **🇺🇸 The Result:** If the IP matches and the time is within business hours, the login is categorized as legitimate rather than a breach.
* **🇧🇷 O Resultado:** Se o IP coincidir e o horário estiver dentro do expediente, o login é categorizado como legítimo em vez de uma invasão.

## 🛡️ Insight do Analista | Analyst's Insight

**🇺🇸 English:**
Aggregation is about "having" the data; correlation is about "understanding" it. Without **Normalization**, your **team** would waste hours comparing different log formats from various vendors. Tools like **Chronicle** do this automatically, allowing you to focus on the high-level relationship between a firewall block and a failed IAM login.

**🇧🇷 Português:**
*Agregação é sobre "ter" os dados; correlação é sobre "entendê-los". Sem a **Normalização**, sua **equipe** perderia horas comparando diferentes formatos de log de vários fornecedores. Ferramentas como o **Chronicle** fazem isso automaticamente, permitindo que você se concentre na relação de alto nível entre um bloqueio de firewall e um login de IAM falho.*

### 🚀 Glossário Técnico | Technical Glossary

* **🇺🇸 Aggregation:** Gathering diverse data into a single, central repository.
* **🇧🇷 Agregação:** Reunir dados diversos em um único repositório central.
* **🇺🇸 Correlation:** Drawing connections between separate events to identify patterns.
* **🇧🇷 Correlação:** Traçar conexões entre eventos separados para identificar padrões.
* **🇺🇸 Normalization:** Standardizing data formats for consistency and faster analysis.
* **🇧🇷 Normalização:** Padronizar formatos de dados para consistência e análise mais rápida.

**"Data is the soil; aggregation is the harvest; correlation is the nourishment."**
*(Dados são o solo; agregação é a colheita; correlação é o alimento.)*
