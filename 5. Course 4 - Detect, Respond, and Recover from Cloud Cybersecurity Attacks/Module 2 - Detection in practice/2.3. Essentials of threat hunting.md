


# 🏹 Threat Hunting e Framework MITRE ATT&CK

**🇺🇸 English:** Threat hunting is a proactive method used to identify previously unknown threats within a network. It acts like a digital detective, searching for clues and pivot points where attackers might be hiding to remain undetected.
**🇧🇷 Português:** O *Threat Hunting* é um método proativo usado para identificar ameaças anteriormente desconhecidas dentro de uma rede. Ele atua como um detetive digital, buscando pistas e pontos de articulação onde os atacantes podem estar escondidos para permanecerem indetectados.

---

### 1. Benefícios da Caça a Ameaças | Benefits of Threat Hunting

A prática do *Threat Hunting* traz vantagens estratégicas para a equipe de segurança:

* **🇺🇸 Mitigate Threats:** Identifies issues early to prevent further damage.
* **🇧🇷 Mitigar Ameaças:** Identifica problemas precocemente para evitar danos maiores.
* **🇺🇸 Faster Response:** Actively analyzes traffic and gathers data to improve defenses.
* **🇧🇷 Resposta mais Rápida:** Analisa ativamente o tráfego e coleta dados para melhorar as defesas.
* **🇺🇸 Inform Business Objectives:** Provides insights into the security state to align with organizational priorities.
* **🇧🇷 Informar Objetivos de Negócio:** Fornece *insights* sobre o estado da segurança para alinhar com as prioridades organizacionais.

---

### 2. Framework MITRE ATT&CK e TTPs

O framework **MITRE ATT&CK** fornece uma linguagem comum para entender ataques através de **TTPs** (Táticas, Técnicas e Procedimentos).

| Componente | 🇺🇸 English (Concept) | 🇧🇷 Português (Conceito) | 💡 Exemplo / Example |
| --- | --- | --- | --- |
| **Tactics** | **Táticas** | O "porquê" ou motivo do atacante. | **Initial Access:** Ganhar acesso à rede. |
| **Techniques** | **Técnicas** | O "como" ou ações específicas realizadas. | **Exploit Public-Facing App:** Usar uma falha pública. |
| **Procedures** | **Procedimentos** | A implementação específica da técnica. | **SQL Injection:** Uso de um comando específico. |

### 3. Ferramentas e Recursos | Tools and Features

O Google Cloud utiliza o **Chronicle SIEM** para potencializar o *Threat Hunting* integrando o mapeamento do MITRE ATT&CK.

* **🇺🇸 Automated Detection:** Uses AI and Machine Learning to reduce manual effort.
* **🇧🇷 Detecção Automatizada:** Usa IA e Aprendizado de Máquina para reduzir o esforço manual.
* **🇺🇸 Pattern Recognition:** Identifies subtle or complex attack patterns in data.
* **🇧🇷 Reconhecimento de Padrões:** Identifica padrões de ataque sutis ou complexos nos dados.
* **🇺🇸 Historical Analysis:** Analyzes past activity to find latent threats.
* **🇧🇷 Análise Histórica:** Analisa atividades passadas para encontrar ameaças latentes.

### 🚀 Glossário Técnico | Technical Glossary

* **🇺🇸 TTPs (Tactics, Techniques, and Procedures):** The specific behaviors of a threat actor.
* **🇧🇷 TTPs:** Os comportamentos específicos de um agente de ameaça.
* **🇺🇸 Pivot Points:** Locations in a network used by attackers to move laterally.
* **🇧🇷 Pontos de Articulação:** Locais em uma rede usados por atacantes para se moverem lateralmente.
* **🇺🇸 Zero-day:** A vulnerability unknown to the software creator.
* **🇧🇷 Zero-day:** Uma vulnerabilidade desconhecida pelo criador do software.

**"In cybersecurity, staying proactive is key, and threat hunting is the bridge between defense and offense."**
*(Em cibersegurança, manter-se proativo é fundamental, e o threat hunting é a ponte entre a defesa e o ataque.)*
