



# 🔍 Consultas Eficazes: RegEx e UDM | Effective Querying: RegEx & UDM

**🇺🇸 English:** Sorting through large amounts of aggregated data is a vital skill for investigations. Tools like Regular Expressions (RegEx) and the Unified Data Model (UDM) allow analysts to filter, sort, and manipulate log data to identify threats quickly.
**🇧🇷 Português:** Filtrar grandes quantidades de dados agregados é uma habilidade vital para investigações. Ferramentas como Expressões Regulares (RegEx) e o Modelo de Dados Unificado (UDM) permitem que os analistas filtrem, ordenem e manipulem dados de log para identificar ameaças rapidamente.

---

### 1. Expressões Regulares (RegEx)

**🇺🇸 English:** RegEx is a sequence of characters that forms a pattern to search text. It offers increased search performance compared to regular text queries.
**🇧🇷 Português:** RegEx é uma sequência de caracteres que forma um padrão para pesquisar texto. Oferece um desempenho de pesquisa superior em comparação com consultas de texto comuns.

#### **Fundamentos de RegEx | RegEx Fundamentals:**

* **`^` (Caret):** 🇺🇸 Matches the beginning of a string. | 🇧🇷 Corresponde ao início de uma string.
* **`[ ]` (Brackets):** 🇺🇸 Matches a range of characters (e.g., `[abc]`). | 🇧🇷 Corresponde a um intervalo de caracteres.
* **`.` (Period):** 🇺🇸 Wildcard that matches any single character. | 🇧🇷 Caractere curinga que corresponde a qualquer caractere único.
* **`\s`:** 🇺🇸 Matches any whitespace (space or tab). | 🇧🇷 Corresponde a qualquer espaço em branco.
* **`\d`:** 🇺🇸 Matches any digit (0-9). | 🇧🇷 Corresponde a qualquer dígito.

---

### 2. Unified Data Model (UDM)

**🇺🇸 English:** Chronicle formats aggregated data using UDM, a consistent data model that organizes log data like a labeled filing cabinet. This makes data easier to search and interpret across different sources.
**🇧🇷 Português:** O Chronicle formata dados agregados usando o UDM, um modelo de dados consistente que organiza os logs como um arquivo etiquetado. Isso facilita a pesquisa e interpretação de dados de diferentes fontes.

* **🇺🇸 UDM Search:** Allows searching through ingested security data that has been normalized and formatted.
* **🇧🇷 Pesquisa UDM:** Permite pesquisar dados de segurança ingeridos que foram normalizados e formatados.

---

### 3. Exemplo Prático: Regra de Login em Cidades Diferentes

Uma regra **YARA-L** no Chronicle pode usar UDM e lógica de correspondência para detectar ataques como o "Impossible Travel":

* **🇺🇸 Events:** Looks for user logins and associated cities.
* **🇧🇷 Eventos:** Procura por logins de usuários e cidades associadas.
* **🇺🇸 Match:** Identifies a user logged in within a **5-minute period**.
* **🇧🇷 Correspondência:** Identifica um usuário logado em um **período de 5 minutos**.
* **🇺🇸 Condition:** Triggers an alert if the user logs in from **more than one city** in that timeframe.
* **🇧🇷 Condição:** Dispara um alerta se o usuário logar de **mais de uma cidade** nesse intervalo de tempo.

## 🛡️ Insight do Analista | Analyst's Insight

**🇺🇸 English:**
Mastering RegEx is a superpower for a security analyst. Instead of scrolling through 10,000 lines of SSH logs, a query like `logName =~ "^SSH"` instantly narrows your focus. When combined with **UDM**, you stop searching for "raw text" and start searching for "security events," which is the key to reducing the time to detect a breach.

**🇧🇷 Português:**
*Dominar RegEx é um superpoder para um analista de segurança. Em vez de rolar por 10.000 linhas de logs SSH, uma consulta como `logName =~ "^SSH"` foca sua atenção instantaneamente. Quando combinado com o **UDM**, você para de pesquisar por "texto bruto" e começa a pesquisar por "eventos de segurança", que é a chave para reduzir o tempo de detecção de uma invasão.*

### 🚀 Glossário de Consultas | Querying Glossary

* **🇺🇸 String:** A sequence of characters, such as "SSH" or an IP address.
* **🇧🇷 String:** Uma sequência de caracteres, como "SSH" ou um endereço IP.
* **🇺🇸 Syntax:** The specific set of rules that defines the combinations of symbols in a language.
* **🇧🇷 Sintaxe:** O conjunto específico de regras que define as combinações de símbolos em uma linguagem.
* **🇺🇸 YARA-L:** A query language used in Chronicle to create detection rules.
* **🇧🇷 YARA-L:** Uma linguagem de consulta usada no Chronicle para criar regras de detecção.
