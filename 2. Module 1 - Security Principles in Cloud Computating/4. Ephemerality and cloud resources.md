In the cloud, change is the only constant. Cloud computing refers to the practice of using **on-demand** resources hosted over the internet, reducing IT overhead while enhancing scalability and responsiveness.

### ‚è±Ô∏è The Concept of Ephemerality
**Ephemerality** is the idea that things exist only for a short period.
* **Instances:** Server resources that run workloads. They can be created or retired quickly.
* **Security Advantage:** Ephemeral resources are harder to target. Attackers must find and exploit vulnerabilities before the resource is destroyed and replaced by a fresh, healthy one.
* **Practice:** Instead of patching an old server, analysts simply launch a new, updated server image.

### üß© Core Cloud Resources
1. **Compute:** The "brainpower" (CPU, Memory). Runs workloads on remote servers without the need for physical equipment.
2. **Storage:** Keeps data accessible on off-site, cloud-based devices managed by the CSP.
3. **Network:** Unlike traditional hardware-based networks, cloud networks are **software-defined**. This allows for quick design changes and total visibility of incoming/outgoing traffic.

> **Example:** A retailer can scale its **Compute** power up during a big sale to avoid website lag (low latency) and scale back down when the peak ends to save costs.
