   - B) Uptime
   - C) Redundancy

### üìù Detailed Explanations (English)

* **Q1 - Correct Answers: C and D:** Cloud storage eliminates upfront hardware costs (**Savings**) and allows for massive expansion without buying physical disks (**Scalability**). Options A and B describe the *on-premises* model.
* **Q2 - Correct Answer: B (Structured data):** Data that fits perfectly into tables (rows/columns) like a spreadsheet is called structured data. It is highly searchable.
* **Q3 - Correct Answer: A (Buckets):** Audio files are *unstructured* data. In the cloud (specifically Google Cloud), objects like audio/video are stored in virtual containers called **Buckets** (Object Storage).
* **Q4 - Correct Answer: B (A repository):** Repositories (like GitHub or internal code repos) are the industry-standard central locations for developers to collaborate on and version data.
